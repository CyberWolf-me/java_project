# Quick Start Guide

## You're Using Java 24 - You Need JavaFX!

Since Java 24 doesn't include JavaFX, follow these steps:

### 1. Download JavaFX (2 minutes)

1. Visit: **https://openjfx.io/**
2. Download **JavaFX SDK 21** for Windows
3. Extract to: `C:\javafx-sdk-21` (or any location you prefer)

### 2. Update the Script (30 seconds)

Open `run.ps1` and change line 5:
```powershell
$JAVAFX_PATH = "C:\javafx-sdk-21"  # Change to your path!
```

### 3. Run the Game!

```powershell
.\run.ps1
```

That's it! The game should launch.
